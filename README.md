# infinite-yie1d

Command line for brute force password attacking.

Usage (after install):

```bash
# install locally
pip install .

# or editable during development
pip install -e .

# install from GitHub
pip install git+https://github.com/ne0nclidev/infinite-yie1d.git

# then run the CLI
infinite-yie1d --echo hello
```
